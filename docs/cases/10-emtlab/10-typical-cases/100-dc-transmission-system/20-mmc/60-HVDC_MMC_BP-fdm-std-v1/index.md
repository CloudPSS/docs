## 模型介绍

### 案例说明

双极端对端柔直输电系统案例模拟了一个端对端真双极柔性直流输电系统，包含送、受两端换流站共四个MMC以及两极直流输电线路，送、受端电网由短路比为8的等值电路模拟。

![双极端对端柔直输电系统](./HVDC_MMC_BP-fdm-std-v1.png "双极端对端柔直输电系统")

案例中直流系统的关键参数如下表所示。

| 参数 | 单位 | 值 |
|:----------- |:---- |:---------:|
| 直流额定功率 | MW | 3000 |
| 直流额定电压 | kV | ±800 |
| 直流接线方式 | -- | 双极大地回线 |
| 送端网侧额定电压 | kV | 525 |
| 受端网侧额定电压 | kV | 525 |
