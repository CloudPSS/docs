## 常见问题
**故障清除时刻**选择里的**过零时刻**和**任意时刻**有什么区别

:
    故障清除的本质是模拟理想断路器开断故障电流。两种故障清除时刻的含义如下：
    - **过零时刻**：当理想断路器接收到分闸信号，断路器不会立即断开故障电流，它会等待被其开断的故障电流自然过零点的时刻，在过零点处完成开断，从而实现故障清除。
    - **任意时刻**：在设定的**故障结束时刻**，无视当前电流相位，立即将故障支路开路（理想交流断路器瞬间开断故障电流）。
   
    需要注意的是，如选择了**任意时刻**，并且**故障结束时刻**的故障电流不为零，强行切断电流会产生极高的 $di/dt$，从而在系统电感（特别是变压器漏感、线路电感）上感应出巨大的暂态过电压（$V=L*di/dt$）。  

    建议使用方式：
    - 关心故障过程与故障恢复过程中的电磁暂态特性，建议选择**过零时刻**。
    - 关心故障后的稳定性，如发电机功角、转速和电网频率的变化，可选择**任意时刻**。
