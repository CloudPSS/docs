## 功能介绍  

CloudPSS 是国内第一款具有完全自主知识产权的新一代电磁暂态仿真工具。CloudPSS 平台基于 WEB 技术、云计算和异构并行计算技术、物联网技术等融入能源互联网领域，形成了全新的精准建模分析平台。基于网页的在线仿真设计，使得 CloudPSS 平台更加简约、便捷。与此同时，我们通过构建并运行海量算例，训练人工智能体，不断提高平台进行优化调控与智能决策的能力，实现系统优化能力的“可进步”，从而将先进的分析和决策技术运用于各能源行业，将超强的计算力转化为超越的生产力。  

### 云计算架构  

CloudPSS 采用先进的云计算架构，将便捷的终端用户算例构建与强大的服务器计算能力相结合。用户可随时随地在本地浏览器内构建仿真算例，完毕后算例文件自动上传至仿真计算服务器完成高效率计算，并将仿真结果数据下传至本地浏览器显示窗口。云计算架构使得用户无需进行复杂的安装过程，仿真计算速度不依赖于本地计算器性能，仿真软件的迭代更新也更易于实现统一管理及不同版本算例兼容。  

### 元件模型与算例搭建  

CloudPSS 提供了包括电力电子变流器、电机、阻抗、电源等电气元件以及数学计算、传递函数信号源等控制元件在内的超 150 种元件模型，完全满足用户对任意电力系统拓扑的仿真需求。基于 Matlab、Python 接口的用户自定义元件，基于“系统”功能的多级子系统嵌套构建、基于“参数及引脚体系”的元件参数管理。基于 Dashboard 的用户数据可视化展示、PSASP/BPA/PSCAD/Simulink 算例转换工具、说明文档库、算例库等特色功能极大程度地丰富了用户建模操作。  

### 建模仿真算法与并行计算  

CloudPSS 针对电磁暂态仿真底层的建模技术、计算技术等进行了开创性改进，提出了诸如分段平均化建模技术、移频分析法、多时间尺度建模技术、开关预判快速化建模、戴维南等效建模等系列先进建模技术，以及包括异构并行架构下批量化电磁暂态仿真并行算法在内的诸多核心算法程序。与此同时，基于开放框架下的仿真调用接口使得用户可以通过 SDK 创建自己的高级仿真应用，从而实现对多场景、多业务的仿真计算能力支撑。相关技术的应用使得 CloudPSS 在仿真效率、仿真精度和仿真应用场景方面实现了对传统电磁暂态仿真工具的超越。  

### 数字孪生体构建  

CloudPSS 平台可通过海量仿真算例训练，完成对仿真系统的优化调控与智能决策，实现平台的可迭代进步。通过构建与实物系统完全相同的“数字孪生”模型，并结合泛在设备传感器的数据反馈，可形成全新的精准建模分析方案和智能运维决策方案将提供强大的分析和决策能力，提高能源网络智能化水平和运营效率，实现现代信息技术与传统能源行业之间的深度融合。  

### 开放式协同仿真  

基于 CloudPSS 发展的 CloudPSS-RT 是工业级的电力系统实时仿真设备，可应用于实时仿真、控制、测试以及其它相关领域 CloudPSS-RT 包含了基于 CloudPSS 的网页仿真操作界面及高性能实时仿真目标机，用户可以快速地使用 CloudPSS 进行电力/控制系统拓扑构建并利用目标机进行实时仿真运算及输出。目标机支持高速 Al /DI/AO/ DO 类的输入输出接口，可用于用户进行硬件在环仿真及快速控制原型开发。CloudPSS-RT 提供了多路 SFP 接口，可实现多台实时仿真器的协同仿真及组网，以及实现与 RT-Lab/RTDS/ADPSS 等实时仿真器的协同仿真。
