系统设置

系统设置页面允许用户对设备进行时间配置、重启和关机操作。

时间设置​

按照以下步骤执行操作：

  1. 点击时间设置按钮。

  2. 在手动配置栏选择日期和时间，点击应用，手动设置设备日期时间。

  3. 或者在 NTP 服务配置 栏输入授时服务器地址，点击应用同步授时服务器时间。
