元件定义​

元件说明​

PSASP_GOV_8元件参数标签页包括属性、参数、引脚三类参数，下面对每类参数进行详细说明。

属性​

CloudPSS 提供了一套统一的元件属性功能，关于元件属性参数的配置，详见 参数卡 页面。
