三绕组变压器参数转换流程

PSASP 三绕组变压器参数​

PSASP 三绕组变压器原始参数​

  1. 绕组 1 额定容量 S1NS_{1N}S1N​（单位：MVA）

  2. 绕组 2 额定容量 S2NS_{2N}S2N​（单位：MVA）

  3. 绕组 3 额定容量 S3NS_{3N}S3N​（单位：MVA）

    三绕组变压器额定容量 SNS_NSN​（单位：MVA）定义为最大绕组的绕组容量，即 SN=max⁡[S1N,S2N,S3N]{S_N} = \max \left[ { {S_{1N} },{S_{2N} },{S_{3N} } } \right]SN​=max[S1N​,S2N​,S3N​]

  4. 额定线电压 Vi,NV_{i,N}Vi,N​（单位：kV，i=1,2,3i=1,2,3i=1,2,3 分别代表绕组 1、绕组 2 和绕组 3）

  5. 由分接头位置计算得出的额定线电压 Vi,actV_{i,act}Vi,act​（单位：kV，i=1,2,3i=1,2,3i=1,2,3 分别代表绕组 1、绕组 2 和绕组 3）

    分接头电压计算
    Vi,act=Vi,N+Vi,NVi,step%(Posi,N−Posi,act)/100{ V_{i,act} } = {V_{i,N} } + {V_{i,N} }{V_{i,step}\%}\left( { {Pos_{i,N} } - {Pos_{i,act} } } \right) / 100Vi,act​=Vi,N​+Vi,N​Vi,step​%(Posi,N​−Posi,act​)/100

    其中，Posi,NPos_{i,N}Posi,N​ 为额定电压下主接头位置；Posi,actPos_{i,act}Posi,act​ 为实际分接头位置；Vi,step%V_{i,step}\%Vi,step​% 为连续两个分接头之间的电压变化率。

  6. 1-2 绕组短路试验损耗（未折算）ΔP12\Delta P_{12}ΔP12​（单位：kW）

  7. 2-3 绕组短路试验损耗（未折算）ΔP23\Delta P_{23}ΔP23​（单位：kW）

  8. 3-1 绕组短路试验损耗（未折算）ΔP31\Delta P_{31}ΔP31​（单位：kW）

  9. 1-2 绕组短路试验电压百分比（折算后）V12%V_{12}\%V12​%（单位：p.u.）

  10. 2-3 绕组短路试验电压百分比（折算后）V23%V_{23}\%V23​%（单位：p.u.）

  11. 3-1 绕组短路试验电压百分比（折算后）V31%V_{31}\%V31​%（单位：p.u.）

此处，折算是指将试验数据（短路损耗、短路电压百分比）折算到统一容量下。对于绕组容量不一致的情况，先将未折算的短路损耗数据折算到统一容量 SNS_NSN​ 基准下。

ΔPS12=ΔP12(SNmin⁡[S1N,S2N])2\Delta {P_{S12} } = {\Delta P_{12} }{\left( {\frac{ { {S_N} } }{ {\min \left[ { {S_{1N} },{S_{2N} } } \right]} } } \right)^2}ΔPS12​=ΔP12​(min[S1N​,S2N​]SN​​)2 ΔPS23=ΔP23(SNmin⁡[S2N,S3N])2\Delta {P_{S23} } = {\Delta P_{23} }{\left( {\frac{ { {S_N} } }{ {\min \left[ { {S_{2N} },{S_{3N} } } \right]} } } \right)^2}ΔPS23​=ΔP23​(min[S2N​,S3N​]SN​​)2 ΔPS31=ΔP31(SNmin⁡[S1N,S3N])2\Delta {P_{S31} } = {\Delta P_{31} }{\left( {\frac{ { {S_N} } }{ {\min \left[ { {S_{1N} },{S_{3N} } } \right]} } } \right)^2}ΔPS31​=ΔP31​(min[S1N​,S3N​]SN​​)2

进而求得每个绕组的短路损耗，

ΔPS1=12(ΔPS12+ΔPS31−ΔPS23)\Delta {P_{S1} } = \frac{1}{2}\left( {\Delta {P_{S12} } + \Delta {P_{S31} } - \Delta {P_{S23} } } \right)ΔPS1​=21​(ΔPS12​+ΔPS31​−ΔPS23​) ΔPS2=12(ΔPS12+ΔPS23−ΔPS31)\Delta {P_{S2} } = \frac{1}{2}\left( {\Delta {P_{S12} } + \Delta {P_{S23} } - \Delta {P_{S31} } } \right)ΔPS2​=21​(ΔPS12​+ΔPS23​−ΔPS31​) ΔPS3=12(ΔPS23+ΔPS31−ΔPS12)\Delta {P_{S3} } = \frac{1}{2}\left( {\Delta {P_{S23} } + \Delta {P_{S31} } - \Delta {P_{S12} } } \right)ΔPS3​=21​(ΔPS23​+ΔPS31​−ΔPS12​)

同理，可求得每个绕组的短路电压百分比

V1%=12(V12%+V31%−V23%){V_1}\% = \frac{1}{2}\left( { {V_{12} }\% + {V_{31} }\% - {V_{23} }\% } \right)V1​%=21​(V12​%+V31​%−V23​%) V2%=12(V12%+V23%−V31%){V_2}\% = \frac{1}{2}\left( { {V_{12} }\% + {V_{23} }\% - {V_{31} }\% } \right)V2​%=21​(V12​%+V23​%−V31​%) V3%=12(V23%+V31%−V12%){V_3}\% = \frac{1}{2}\left( { {V_{23} }\% + {V_{31} }\% - {V_{12} }\% } \right)V3​%=21​(V23​%+V31​%−V12​%)
  1. 空载试验损耗 P0P_0P0​（单位：kW）
  2. 空载试验电流 I0%I_0\%I0​%（单位：p.u.）

PSASP 三绕组变压器有名值参数​

由原始参数可计算得到三绕组变压器有名值参数：

  1. iii 侧正序漏电阻 RTiR_{Ti}RTi​（单位：ohm），归算到绕组 iii 侧，即 RTi=ΔPSi1000SNVi,act2SN{R_{Ti} } = \frac{ {\Delta {P_{Si} } } }{ {1000{S_N} } }\frac{ {V_{i,act}^2} }{ { {S_N} } }RTi​=1000SN​ΔPSi​​SN​Vi,act2​​
  2. iii 侧正序漏电抗XTiX_{Ti}XTi​（单位：ohm），归算到绕组 iii 侧，即 XTi=Vi%100Vi,act2SN{X_{Ti} } = \frac{ { {V_i}\% } }{ {100} }\frac{ {V_{i,act}^2} }{ { {S_N} } }XTi​=100Vi​%​SN​Vi,act2​​
  3. 励磁电导 GMG_MGM​（单位：mho），归算到绕组 1 侧，即 GM=ΔP01000V1,N2{G_M} = \frac{ {\Delta {P_0} } }{ {1000V_{1,N}^2} }GM​=1000V1,N2​ΔP0​​
  4. 励磁电纳 BMB_MBM​（单位：mho），归算到绕组 1 侧，即 BM=I0%100SNV1,N2{B_M} = \frac{ { {I_0}\% } }{ {100} }\frac{ { {S_N} } }{ {V_{1,N}^2} }BM​=100I0​%​V1,N2​SN​​

PSASP 三绕组变压器标幺值参数（潮流计算用）​

利用上述 PSASP 双绕组变压器原始参数，计算系统基值下的标幺值参数（系统功率基值 SbS_bSb​，各侧电压基值 Vi,bV_{i,b}Vi,b​）

  1. 正序漏电阻RTi∗R_{Ti}^*RTi∗​ （单位：p.u.） RTi∗=ΔPSi1000SNVi,act2SNSBVi,B2{R_{Ti}^*} = \frac{ {\Delta P_{Si} } }{ {1000{S_N} } }\frac{ {V_{i,act}^2} }{ { {S_N} } }\frac{ { {S_B} } }{ {V_{i,B}^2} }RTi∗​=1000SN​ΔPSi​​SN​Vi,act2​​Vi,B2​SB​​
  2. 正序漏电抗XTi∗X_{Ti}^*XTi∗​ （单位：p.u.） XTi∗=Vi%100Vi,act2SNSBVi,B2{X_{Ti}^*} = \frac{ {V_i\% } }{ {100} }\frac{ {V_{i,act}^2} }{ { {S_N} } }\frac{ { {S_B} } }{ {V_{i,B}^2} }XTi∗​=100Vi​%​SN​Vi,act2​​Vi,B2​SB​​
  3. 励磁电导GM∗G_M^*GM∗​ （单位：p.u.） GM∗=ΔP01000Vi,N2Vi,B2SB{G_M^*} = \frac{ {\Delta {P_0} } }{ {1000V_{i,N}^2} }\frac{ {V_{i,B}^2} }{ { {S_B} } }GM∗​=1000Vi,N2​ΔP0​​SB​Vi,B2​​
  4. 励磁电纳BM∗B_M^*BM∗​ （单位：p.u.） BM∗=I0%100SNVi,N2Vi,B2SB{B_M^*} = \frac{ { {I_0}\% } }{ {100} }\frac{ { {S_N} } }{ {V_{i,N}^2} }\frac{ {V_{i,B}^{\text{2} } } }{ { {S_B} } }BM∗​=100I0​%​Vi,N2​SN​​SB​Vi,B2​​
  5. 三侧非标准变比Tki∗T_{ki}^*Tki∗​ （单位：p.u.） Tki∗=Vi,actVi,B{T_{ki}^*} = \frac{ { {V_{i,act} } } }{ { {V_{i,B} } } }Tki∗​=Vi,B​Vi,act​​
提示

在考虑分接头影响时，CloudPSS 中计算有名值和标幺值计算与《PSASP7.3 版图模一体化平台用户手册》中所提供的计算方式略有区别。CloudPSS 中，分接头仅影响漏阻和漏抗，不影响励磁导纳。如有疑问请联系 songyankan@cloudpss.net
