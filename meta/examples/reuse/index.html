重用内容

导入内容：

import Hello from './_hello-comp.md';
import Comp from './_comp-comp.md';

以下为来自组件的内容：

<Hello name="Alice" />

这是一个 MarkDown 组件。

你好，Alice！

使用默认参数：

<Hello />

这是一个 MarkDown 组件。

你好，用户！

传递内容：

<Hello>
  :::note
  内容
  :::
</Hello>

这是一个 MarkDown 组件。

你好，用户！

备注

内容

组件嵌套：

<Comp compName={frontMatter.title}/>

传递的参数：

{
  "compName": "重用内容"
}

在组件内使用组件，并传递参数：

这是一个 MarkDown 组件。

你好，重用内容！
