Server Manager 文档

产品许可证

产品许可证页面展示从 CloudPSS 服务获取的全部产品许可，允许用户上传、查看证书。页面如下：

子服务器管理

子服务器管理页面展示当前与本机组网的全部子服务器，允许用户删除子服务器。页面显示如下：

容器管理

容器管理页面支持查看、关闭服务器上与 CloudPSS 相关的容器应用。

平台升级

平台升级页面支持查看 CloudPSS 包版本，以及对包进行安装，升级。平台升级页面包括可用更新和源列表两部分组成。

服务器组网

服务器组网页面会默认发现当前局域网内全部 CloudPSS 服务器并允许用户自行组网。页面显示如下：

服务状态

服务状态页面显示服务器上所有与 CloudPSS 相关的软件服务运行状态，允许对相应服务执行启动、重启和关闭操作。

状态概览

状态概览页面展示服务器设备的实时运行状况和基本信息，涵盖从设备出厂时的初始配置到当前的工作负载等系统关键数据。

登录

有以下几种方式均可以访问 CloudPSS Server Manager 服务。

系统备份

系统备份页面允许用户对 CloudPSS 全部服务进行备份管理，支持手动备份（下载、删除、锁定、还原系统备份）和自动定时备份。

系统设置

系统设置页面允许用户对设备进行时间配置、重启和关机操作。

网络配置

网络配置模块允许对设备的网络连接进行设置。页面如下：

运维

运维页面允许用户开启系统运维模式（默认关闭状态），运维模式开启后获得一个运维二维码，将这个二维码截图发给 CloudPSS 的工程师，他们就可以通过互联网连接机器进行远程维护。

配置管理

配置管理模块提供一个集中的界面，允许用户能够轻松地管理和配置 CloudPSS 各个软件及其服务。所有服务的配置项和设置通过配置定义文件描述，并通过直观的图形界面进行配置和管理。
